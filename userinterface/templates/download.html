TRY it!

Download the Omero Plugin at the bottom of this page, and try the Rhadius demo server.

Requirements

Rhadius plugin requires the following libraries to be installed on your OMERO server installation:

  • numpy
  • requests
  • PIL
  • time

Install

Upload the plugin using OMERO.insight or copy the plugin into omero server [lib/scripts] folder.

Setup

Load the plugin either from OMERO.insight or OMERO.web and setup the parameters (leave default values to try our test server):

Folder Name:
For each remote analysis available on Rhadius a script will appear inside this folder.

Rhadius Server URL:
OMERO Server must be able ton reach Rhadius Server trough HTTP.

WebDAV Settings:
To pass images from OMERO to Rhadius a WebDAV server is used. Images are uploaded from Omero via username/password and downloaded by Rhadius via direct link. You can easily achieve this setup using OwnCloud.

OMERO Server Settings:
The automagic creation of new processes inside OMERO Server script directory requires Administrative username and password of your OMERO instance passed to the script.

When everything is set up, run the script.

Run!

Reload the script list and you should now find a new folder with all the remote analysis ready to be run. Have fun!
